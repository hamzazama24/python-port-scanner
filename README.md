Python Port Scanner
Overview

This project is a Python-based port scanner developed using the socket module. The scanner analyzes a target host and identifies open and closed ports across common network services. The results were compared with Nmap scans to verify accuracy and demonstrate a practical understanding of network reconnaissance techniques.

Features
Scans a target host for open and closed ports
Uses Python's built-in socket library
Detects commonly used service ports such as:
SSH (Port 22)
HTTP (Port 80)
HTTPS (Port 443)
Displays scan results in a clear format
Lightweight and easy to run

Validation

The scan results were validated against Nmap, producing consistent findings for open services on a public test host. This comparison helped verify the accuracy of the scanner and provided hands-on experience with industry-standard security tools.
